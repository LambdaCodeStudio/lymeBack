Subir o actualizar una imagen

Método: POST
URL: /api/productos/:id/imagen
Headers: Content-Type: multipart/form-data
Body: Form con campo imagen que contiene el archivo


Obtener una imagen

Método: GET
URL: /api/productos/:id/imagen
Response: Imagen binaria con el Content-Type adecuado


Eliminar una imagen

Método: DELETE
URL: /api/productos/:id/imagen
